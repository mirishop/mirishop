# mirishop

미리숍은 상품을 미리 예약하는 서비스를 제공하는 API 입니다.

### Docker 실행방법

- docker-compose.yml 파일은 docker 폴더에 있습니다.

~~~
$ docker-compose up -d
~~~

### 구현 기능 목록

- [ ] 유저
  - [ ] 회원가입 기능
    - [ ] 이메일 인증 회원가입
    - [ ] 비밀번호 암호화
  - [ ] 로그인 / 로그아웃
    - [ ] jwt 토큰 발급
  - [ ] 프로필 수정
    - [ ] 이름 / 프로필 이미지 / 인사말 수정
    - [ ] 비밀번호 업데이트

- [ ] 팔로우
  - [ ] 다른 유저 팔로우 기능
  - [ ] 상호작용 : 팔로우시 뉴스피드에 유저 글/ 댓글/ 좋아요가 있어야함
  - [ ] 팔로우 하는 유저의 소식도 보여줌
  
- [ ] 포스트
  - [ ] 로그인 유저 포스트 작성

- [ ] 뉴스피드 - 팔로우 한 사용자의 포스트 및 소식 확인
  - [ ] 최신순으로 정렬
  - [ ] 내가 작성한 포스트에 있는 댓글/좋아요 확인가능

- [ ] 댓글
  - [ ] 포스트에 댓글 작성 기능
  - [ ] 포스트 및 댓글에 좋아요 기능
